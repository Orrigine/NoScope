# Sk@t3rZ’
**Sk@t3rZ’** est un prototype de Shoot’em Up 3D inversé mêlant Endless Runner et skate, développé sous Unity.
Le joueur incarne un skater poursuivi par une horde de zombies et doit survivre le plus longtemps possible tout en maximisant son score.

## À propos du projet
Genre : Shoot’em Up 3D / Endless Runner
Plateforme : Windows x64
Durée d’une session : ~10 minutes
Contrôles : Clavier / Souris
Moteur : Unity 6000.2.5f1
